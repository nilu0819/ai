# Curser Project

## Calculator

Run interactive mode:

```bash
python "curser project/calculator.py"
```

Evaluate a single expression:

```bash
python "curser project/calculator.py" "(3+4)*5"
```

Supported operators: `+ - * / // % **` and parentheses.

## Calculator App (GUI)

Run the desktop app:

```bash
python "curser project/calculator_app.py"
```
